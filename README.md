# OSM GeoJSON Regions

This repository contains GeoJSON files for geographic regions, such as countries and administrative areas, extracted from OpenStreetMap data.

The data is intended for use in applications, services, visualizations, and research projects that require reusable geographic boundary data in GeoJSON format.

## Data Source

The geographic data is based on OpenStreetMap.
